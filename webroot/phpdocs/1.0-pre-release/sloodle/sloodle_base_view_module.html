sloodle_base_view_module

Description
Description | Descendents | Vars (details) | Methods (details)

The base class for viewing a SLOODLE module.

Should be inherited and overridden. Child class MUST be named "sloodle_view_NAME", where NAME replaces the name of the feature or module. For example, "sloodle_view_controller" or "sloodle_view_course". For modules, the NAME must match the "type" specifier in the "sloodle" table, such as "controller", "distributor", or "presenter".
