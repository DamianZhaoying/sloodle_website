Description

Enrolment Booth 1.0 configuration form.

This is a fragment of HTML which gives the form elements for configuration of a enrolment booth object, v1.0. ONLY the basic form elements should be included. The "form" tags and submit button are already specified outside. The $auth_obj and $sloodleauthid variables will identify the object being configured.
