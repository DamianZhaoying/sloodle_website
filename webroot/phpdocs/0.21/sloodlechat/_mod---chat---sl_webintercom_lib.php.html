sloodle_get_visible_chatrooms_in_course (line 28)

Gets an array of available chatrooms in the specified course.

Chatrooms which are hidden, or which are in a hidden section of the course, are ignored.

  • return: If successful, a numeric array, associating module instance ID's with database record objects of the activity modules. Returns boolean false if an error occurs.
