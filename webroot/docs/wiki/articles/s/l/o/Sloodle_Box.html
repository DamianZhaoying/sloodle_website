The Sloodle Box allows you to create the Sloodle objects you'll need for your class.

There are two advantages to deploying objects this way:

  • You can tell the box which Moodle URL and course you will be using for the class. The Sloodle Box will pass this information to objects it creates, so you don't have to setup each object individually.
  • You can make the box remember where you put all the objects for the class, so you won't have to set them all up again next time.

Before you start

  • You need the Sloodle module installed on your Moodle server.
  • You need to be registered as a teacher or administrator for a course in Moodle.
  • You need permission to create objects on the land you're standing on in Second Life.

How to use it:

Setting the server

1. Drag the Sloodle Box from your inventory to the ground you're standing on.

2. Touch the box, and type “/1 sloodle”.

3. Type your Moodle URL. For example: 

  http://www.sloodle.com

...or 

  http://www.myschool.org/mydepartment/moodle

(Don't add “index.php”, and don't include a slash at the end of the URL.) You'll be able to change this URL later by clicking on the “Set Moodle” button. The Sloodle Box will prompt you to open a web page at your Moodle site give it permission to talk to Moodle. You'll need to login to Moodle (if you haven't already) and click “Authorize”.

(At this point, the server will send a message to your Sloodle Box giving it an authorization key so that it, and any objects it creates, can prove to the server that they are allowed to talk to it.)

Authenticating your avatar

If it's the first time you have used Sloodle, the Sloodle Box will give you a URL to click to link your avatar to your Moodle account.

(If Sloodle already knows who you are, it will skip this step.)

Setting the course

The Sloodle Box will show you a list of course codes for the courses that you are allowed to teach. Click the course you want to use.

(You will be able to change course later on by clicking “Set course” to get the list of course codes back.)

Creating your Sloodle objects

For each course, you can keep a number of saved layouts or “profiles” controlling which Sloodle objects you will use for your class, and where they will go. If you haven't yet saved any profiles, just click “Default”, which will place one copy of each Sloodle object above the Sloodle Box.

Moving things around

If you don't like where the Sloodle Box put your objects by default, you can move them around. If there are any objects that you don't want to use, you can delete them.

Saving the positions of your objects

Having moved things around and put them where you want them, you can save their positions back to a new profile for next time.

1. Touch the Sloodle Box and type “/1 sloodle”. 2. Click “Save as”. 3. Type a name for your new profile, eg. “/1 myprofile”.

Cleaning up

Click “Cleanup”. The objects should disappear.
