Summary

RegEnrol Booth
Allows users in Second Life to enrol their Moodle account in courses on the Moodle site. (This process separate from registration/authentication of avatars. See Registration and Enrolment for more information).

Compatibility

The Enrolment Booth is a sub-component of the Sloodle Set, so please check the Sloodle Set Compatibility information.


Setup

Automatic Configuration

The Enrolment Booth is distributed as part of the Sloodle Set. It is recommended that you use the official Sloodle.org Object Distributor to obtain an appropriate Set for your Sloodle version. Rez and configure the Sloodle Set, and click on the object vector box (the tall one in the middle). Select the "Course Enrolment Booth" option from the dialog that appears, and a Enrolment Booth should be rezzed for you.

The Sloodle Set will automatically configure the Enrolment Booth for you, so it should be ready to use.


Notecard Configuration

If you prefer, you can configure the Enrolment Booth manually using a notecard. It uses the same format as the 'sloodle_config' notecard for a Sloodle Set, so you can use the Sloodle notecard setup page on your Moodle site to generate one. The text from there should be placed into a notecard called "sloodle_config" in the Enrolment Booth's inventory. You will need to reset the scripts before the notecard is detected.

Notecard configuration is useful because:

  • it will not lose its configuration when it is reset
  • it does not require the use of a Sloodle Set every time
  • it can be slightly more secure than automatic configuration


Usage

When the Registration Booth is rezzed, you may use it by clicking the panel which says "Touch to enrol on this course". It will check the associated Moodle site to see if your avatar is already registered. If not, it will report that you need to register your avatar first. You will need to use a Sloodle Registration Booth or a Sloodle Toolbar to register your avatar with the site.

If you are already registered, then the Enrolment Booth will give you a URL to the Moodle site. Follow that URL, and you will be taken to a page where you can confirm or deny that you want to enrol on the course. Note: this is exactly the same as if you tried to to enrol in the course entirely through Moodle. Once this has taken place, you will be able to use any related course-specific Sloodle tools.

At some point, we hope to make the enrolment automatic, so that you will not need to go to the Moodle site at all. However, that is a very complicated process, and certainly any courses requiring payment or security keys will need to be handled differently.
