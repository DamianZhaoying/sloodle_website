Qu'est ce que Sloodle?

Sloodle est un projet Open Source dont le but est d'associer un support d'apprentissage et d'enseignement oriente web (LMS, de l'anglais "Learning Management System", ou VLE pour "Virtual Learning Environment") qui amplifie l'interaction jeu/technologie basee sur les Environnements Virtuels 3D Multi-Utilisateurs(MUVE).
Actuellement, le developpement de Sloodle s'occupe de l'integration de Moodle a Second Life comme le montre l'image suivante.
