Sloodle Notecard Setup Page

From SLIS Second Life Wiki

Sloodle Versions: 0.19, 0.2, 0.21 (?)

  • Note: this page refers to an old version of Sloodle. You might require equivalent information for Sloodle 0.3.


What is it?

The notecard setup page is a Sloodle configuration page (within Moodle) which lets you automatically generate text for your "sloodle_config" notecards. These can be used in object configuration. However, currently only Moodle administrators can access the notecard setup page, so they will need to provide configuration notecards for educators. In newer versions of Sloodle, notecard configuration is STILL possible, but is accessed differently. To generate a notecard, administrators can do this by clicking on the Sloodle Controller activity within their moodle course page. The Notecard Configuration video can be viewed here

Where is it?

These easiest way to get to it is via your Sloodle Module Configuration page. (It is a link called "Create Notecard" near the top.)

You can also access the script directly in your web-browser by going to the following address, relative to your Moodle root: 

/mod/sloodle/sl_setup_notecard.php


So for example, the notecard setup page on Sloodle.org is: 

http://www.sloodle.org/mod/sloodle/sl_setup_notecard.php


Using the page

When you visit the page, you will be shown a list of available courses, with a radio-button beside each one. Locate the course you need to create a configuration notecard for, check the radio-button, and click "Generate notecard text".

On the next page, the text for the notecard will appear in a text-box. You can copy-and-paste this into a "sloodle_config" notecard in Second Life.
