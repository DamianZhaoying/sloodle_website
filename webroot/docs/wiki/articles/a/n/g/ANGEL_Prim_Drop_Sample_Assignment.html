Sample Assignment Setting

Getting started with 3D assignments

This assignment assesses your ability to:

  • Create an Avatar
  • Link it to ANGEL
  • Find the ANGEL Learning Isle sand box
  • Create an object
  • Change its name, color and textures
  • Use the PrimDrop drop box

Log into ANGEL and change your home page

  • Edit Page > Add Components > ANGEL Island

Connecting your avatar

  • Create your avatar account, log into Second Life and find the PrimDrop Assignment Dropbox: http://slurl.com/secondlife/ANGEL%20Learning%20Isle/231/25/21
  • Touch the box to see your Avatar information such as "e8192837-1234-4abc-61526e1e3536" (do not use this sample)
  • Copy this long code
  • Click "Go to page" in the blue box
  • Add the code and avatar name to the ANGEL Learning Isle nugget.

Complete the Assignment in 3D

  • Use the Second Life building tools to create a Cylinder
  • Change the color to red.
  • Name it "My Cone" and add a description.
  • Change the permissions to allow everyone to edit your work.
  • Right click your object and "Take" it to inventory

Submit the Assignment

  • Find it in your inventory using the "Recent Items" tab
  • Touch the PrimDrop drawer - it will recognize you and open
  • Hold down the CTRL key and drag "My Cone" over the drop box. Wait until you see a red fringe around the box and then release the mouse button.

Success: Checking your work

You should hear a radio static sound and see a notice in chat. A record of your submission is now submitted to the ANGEL dropbox and may be graded by your instructor. Your instructor will be able to see your object by opening this drop box.
