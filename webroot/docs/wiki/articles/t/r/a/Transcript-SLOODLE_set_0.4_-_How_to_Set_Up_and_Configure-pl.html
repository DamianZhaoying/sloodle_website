SLOODLE set 0.4 - How to Set Up and Configure

Youtube Setup Video

Hello everyone. Welcome to the SLOODLE newsroom. Witam w redakcji wiadomości Sloodle.

My name is Paul Preibisch in the real world and Fire Centaur in Second Life. W świecie rzeczywistym mam na imię Paul Preibish a Fire Centaur w Second Life.

Today I’m going to show you the first step in configuring SLOODLE tools in Second Life. Dzisiaj pokażę Wam pierwszy etap w konfiguracji narzędzi Sloodle w Second Life.

This involves creating a SLOODLE Controller activity on your Moodle website, rezzing a SLOODLE Set and then configuring the SLOODLE set to connect with the SLOODLE Controller activity. Uruchomimy Kontroler SLOODLE na stronie Moodle, zrezujemy zestaw SLOODLE a nastepnie skonfigurujemy zestaw SLOODLE w celu polaczenia z aktywnym Kontrolerem SLOODLE.

Let’s get started. Zatem zacznijmy.

Here is the SLOODLE 101 test site. Add a SLOODLE Controller to a class. Click on the SLOODLE Sandbox class. You have to be logged in as an administrator or a course creator. Tutaj jest widoczna strona testowa SLOODLE 101. Dodajcie Kontroler SLOODLE do lekcji. kliknijcie na SLOODLE Sandbox class.

Now, in the top right I’m going to Turn Editing On. Teraz, w prawej gornej części przełącze w tryb edycji (Turn Editing On).

Now we can add an activity. Nastepnie możemy dodać aktywność.

Go to the “Add an Activity” drop-down box W rozwijanym menu "Add an Activity"

Select “SLOODLE Controller” under SLOODLE modules. Zaznaczcie prosze "SLOODLE Controller", który znajduje sie pod modulami SLOODLE.

Add a new SLOODLE module. “Module type: SLOODLE Controller” Dodajcie nowy moduł SLOODLE. "Module type: SLOODLE Controller"

Type “SLOODLE Sandbox Controller” and a description. one Wpiszcie "SLOODLE Sandbox Controller" i wprowadźcie opis.

Scroll down the bottom and press “Save and return to the course.” Przewińcie okno na dół i naciśnijcie "Save and return to the course" - zachowaj i powróć do lekcji.

Great! We now have a SLOODLE Controller and we can now connect our SLOODLE set to it in Second Life. Wspaniale! Mamy teraz Kontroler SLOODLE i możemy połączyć z nim zestaw SLOODLE, który jest w Second Life.

We just set up our SLOODLE Controller on our Moodle website. Właśnie ustawiliśmy Kontroler SLOODLE na naszej stronie Moodle.

We need to rez the SLOODLE Set in Second Life and connect it to the Controller. Powinniśmy teraz zrezować zestaw SLOODLE w Second Life i podłączyć go do Kontrolera.

There are two places that you can get the SLOODLE Set: you can either teleport to Sloodle Island and get it from the marketplace, or you can actually go on SL Exchange and search for it there and have it sent to you. Istnieją dwa miejsca, gdzie możecie otrzymać zestaw SLOODLE: możecie teleportować się na wyspe SLOODLE i pobrać zestaw ze stoiska, lub poszukać na stronie SL Exchange.

Go to the SLOODLE marketplace and get it there. So let’s go! Idziemy na stoisko SLOODLE i pobierzemy zestaw z tego miejsca.

Press control F on a PC or command F on a Mac to get the Second Life search window. Naciśnijcie Ctrl F lub polecenie F na Macu w celu otworzenia okna Szukaj w Second Life.

Click the “Places” tab and search for “SLOODLE” Kliknijcie zakładkę "Miejsca" i wpiszcie słowo kluczowe "SLOODLE".

Find Sloodle Island and press “teleport.” Odszukajcie wyspe SLOODLE i wciśnijcie guzik "Teleportuj".

You arrive in the SLOODLE welcome area Przemieszczamy się na wyspe SLOODLE.

Find the directory that shows the Island map. Odszukajcie miejsce, gdzie pokazana jest mapa wyspy.

Click on the option that says, “SLOODLE free-market, drop in for your free SLOODLE tools.” Kliknijcie na opcje z napise, "SLOODLE free-market, drop in for your free SLOODLE tools".

You arrive in the SLOODLE marketplace. Przybywacie na stoisko SLOODLE.

Look through the marketplace and find the SLOODLE set. Rozejrzyjcie się i odszukajcie zestaw SLOODLE.

You don’t want version 0.21. Nie chcecie wersji 0.21.

0.3... 0.3 ...

We want the 0.4 version. Chcemy wersje 0.4.

Click on the vending machine and select the SLOODLE Set from the menu. Kliknijcie na automat do sprzedaźy.

The SLOODLE set is delivered to me. Press “Keep.” Zestaw SLOODLE zostaje Wam dostarczony. Kliknijcie przycisk "Zachowaj".

Look in your inventory in “Recent Items” under “Objects” there is the SLOODLE Set. Otwórzcie Szafe i odszukajcie w zakładce Nowe Obiekty w folderze "Objects" zestaw SLOODLE.

Let’s go back to the newsroom. Wracajmy do redakcji.

Look in my inventory and drag the SLOODLE Set onto the ground. Z Szafy przeciągnijcie zestaw SLOODLE na ziemię.

The SLOODLE Set needs to be set to the correct group of the land. Zestaw SLOODLE powinien być ustawiony na prawidłową grupę posiadającą daną parcelę.

Sometimes land is set to only allow group members to run scripts. Here the groups are correct. Czasami ustawienia parceli nie pozwalają na używanie skrypów osobom spoza grupy. Tutaj grupy są prawidłowe.

Touch the podium. Dotknijcie podium.

It says in local chat, “Please check the address of your MOODLE site without the /” Podium mówi w czacie lokalnym "Please check the address of your MOODLE site without the /"

Type in your Moodle site address. Wpiszcie adres strony Moodle.

The SLOODLE set is connecting to my webpage. Click “Go to page” on the menu. Zestaw SLOODLE łączy się z moją stroną internetową. Kliknijcie "Go to page" w menu.

A webpage loads, “SLOODLE Object Authorization / Object Details / Object name: SLOODLE set” Strona ładuje się, “SLOODLE Object Authorization / Object Details / Object name: SLOODLE set”

Scroll down, “SLOODLE Object Authorization / Select Controller” Zjedzcie w dół strony, “SLOODLE Object Authorization / Select Controller”

Select the ”Sandbox Controller” that we created earlier. Zaznaczcie "Sandboc Controller", który utworzylismy wcześniej.

Set “Do you want to authorize this object” to “yes” Ustawcie “Do you want to authorize this object” na “yes”

Press the “submit” button. Kliknijcie guzik "submit"

In green it says “Object authorization has been successful.” Zobaczycie zielony napis “Object authorization has been successful.” - Autoryzacja obiektu powiodła się.

Close this window Zamknijcie to okno.

In Second Life, you will need to download that configuration. Teraz należy skonfigurować zestaw w Second Life.

Click on the podium again, the menu comes up. Kliknijcie ponownie na podium, pojawi się menu.

“0=access web configuration page” would take us to the page we were just at. “0=access web configuration page” poprowadzi nas do strony internetowej, na której właśnie bylismy.

Click on “1=download configuration” kliknijcie na “1=download configuration”

These SLOODLE set will now contact my new website and download what we just authorized. Teraz SLOODLE będzie kontaktowało się z moją nową stroną i załaduje to co właśnie zostało przez nas autoryzowane.

Its ready! Now we can rez items by clicking on this large item that looks like a refrigerator. Gotowe! Teraz możemy zrezować narządzia klikając większy obiekt, który wygląda jak lodówka.

The SLOODLE set is now configured. Zestaw SLOODLE jest już skonfigurowany.

Rezzing the SLOODLE tools Rezowanie narzędzi SLOODLE

Click on the big box part to see a menu and scroll to see the different tools that can be rezzed. Kliknijcie na większym obiekcie, rozwinie się menu i zobaczycie różne narządzia, które mogą być zrezowane.

Click on the “Next page” button. Kliknijcie na przycisk "Next page"

Let’s rez the WebIntercom by pressing number 14. Zrezujmy WebIntercom klikając na numer 14.

An object appears behind the set. This is the WebIntercom. Obiekt pojawi się za zestawem. To jest WebIntercom.

Move the item to a convenient place. Przesuńcie obiekt w odpowiednie miejsce.

Because the item has been rezzed by the SLOODLE Set, we can easily configure it just by clicking on it. Ponieważ obiekt został zrezowany przez Zestaw SLOODLE możemy łatwo skonfigurować go poprzez kliknięcie.

The blue menu sends us to the same configuration webpage. Niebieskie menu kieruje nas do tej samej strony z konfiguracją.

The object configuration page says, “This object has already been authorized. If you want to re-authorize it, then please delete the authorization entry from your SLOODLE controller.” Strona konfiguracji obiektu informuje "“This object has already been authorized. If you want to re-authorize it, then please delete the authorization entry from your SLOODLE controller.” - Ten obiekt został już autoryzowany. Jeśli chcesz ponownie go autoryzować skasuj autoryzację z Kontrolera SLOODLE.

I’m ignoring this warning because I already configured a WebIntercom before with this class. Ignoruje ostrzeżenie ponieważ już skonfigurowałem WebIntercom wcześniej do tej lekcji.

Select the chat room to connect the WebIntercom to and press “Submit.” Zaznaczcie chat room w celu połączenia z WebIntercom i wciśnijcie "Submit"

The site shows some details. Press “Continue.” Strona pokazuje szczegóły, wciśnijcie "Continue"

Back in Second Life Wracamy do Second Life.

Click on the WebIntercom and select “Download configuration” to make it ready. Kliknijcie na WebIntercom i zaznaczcie "Dowload configuration".

You will need to do this with each tool that you now rez from the SLOODLE Set. Będziemy potrzebowali tej konfiguracji dla każdego narzędzia, które teraz będzie zrezowane przez Zestaw SLOODLE.

The routine is the same: rez is the object, click on it, go to the webpage, authorize it and download the configuration. Procedura jest podobna - zrezujcie obiekt, kliknijcie na niego, przejdzcie do stony, autoryzujcie obiekt i pobeirzcie konfiguracje.

It will take a few times to remember the process, but it should be easy with practice. Po kilku razach zapamiętacie proces, praktyka czyni mistrza.

Goodbye. Do widzenia.
