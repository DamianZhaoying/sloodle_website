Translating on the Sloodle wiki

From SLIS Second Life Wiki

The purpose of a wiki is to allow anybody to contribute. Unfortunately, we sometimes have problems with spam, so there are some restrictions in place. After you have created an account here on the wiki, wait for at least 24 hours. If you still cannot edit or add pages, then contact us on the forums at Sloodle.org.

What to translate

Eventually, we hope that the complete documentation for Sloodle will be available in multiple languages. However, Sloodle is a continually developing project, so it the documentation will probably never be 'complete'. Translation work for any part of Sloodle is welcome though.


Page names

When you translate a wiki page, it will need to have a different title, so that it does intefere with the original page. There are two ways of doing this (which may be combined if necessary):

  • Add a language suffix (such as '-es' for Spanish)
  • Translate the page name

In some cases, it is most appropriate to only translate the page name. For example, the Spanish version of the Sloodle homepage is called Sloodle Portada. It is not appropriate to translate page names to other languages when the name directly references an object or component within Sloodle. For example, the 'sloodle_config' page refers to the configuration notecard which is used setup Sloodle objects in-world. The name of this notecard will always be 'sloodle_config', regardless of language, so the wiki page name should not be translated.


How to edit a page

If somebody has already started translating a page, and you want to help, simply click the "edit" tab near the top. (You will need to be signed-in for this to work). Please ensure to use proper wiki formatting when translating pages.


How to create a page

If you want to start a new translation of a page, then you first need to create the page. This can be done in two ways. First, you can create a link on another page, save that page, then click the link you created. You will be automatically asked to edit the new page. Alternatively, you can edit the URL displayed in your browser's address bar -- replace the name of the current page with the name of the new page you want to create. Again, you will automatically be asked to edit the new page.

If a page you want to create already has content, then please do not delete it unless you are absolutely sure you know it is safe to do so. We can reverse most changes if you make a mistake, but please be careful anyway!


Starting a new translation

If you want to start a new translation (i.e. no translation work has been done to your language before) then it is best to contact us on the Sloodle.org forums. After that, you will first want to create a new Sloodle home page for your language, and then start building your translation from there. Remember that all pages should be linked from somewhere. That is, you should not create isolated pages, otherwise nobody will be able to find them.
