Integrate whiteboard module

From SLIS Second Life Wiki

Summary

Make it possible for teachers and students to: a) Edit a shared whiteboard using their web browser b) See the shared whiteboard in SL c) Click on the shared whiteboard to open it in their web browser so that they can see it better and/or edit it. d) (ideally) change the whiteboard contents in SL.

How to do it (easy but crude)

Find whichever whiteboard solution currently works best with Moodle - I've tried Covcell's whiteboard: http://www.covcell.org/index.php?option=com_content&view=article&id=68&Itemid=58 ...which uses Drawboard. http://sourceforge.net/projects/drawboard/ This works OK, although you need to run Java on the server, and the client and server both need an extra port open, which is potentially troublesome...

Figure out a way to save images from the whiteboard back to the server. (I found if I updated the drawboard code in Covcell's thing you could do this, although it's a bit laborious.)

Make an object in SL to set and display video for the parcel, and have that video update regularly based on the most recent image saved on the server.

Edmund Edgar, 2007-05-09
