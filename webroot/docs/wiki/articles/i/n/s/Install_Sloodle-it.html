Accedi alla tua installazione Moodle

  • Usando FTP: Molto probabilmente hai bisogno di un software FTP come ad esempio FileZilla, Dreamweaver o Fugu
  • Se lavori localmente: Se il tuo Moodle è installato sul tuo PC locale, apri semplicemente un gestore dei file di sistema (e.g. "???Sistema" su Windows)
  • Usendo gli strumenti del tuo provider: Fornitori di siti come Dreamhost, Bluehost o Freehostia offrono dei pannelli di controllo basati su pagine web per accedere ai file

Ordina le cartelle unzippate dal file di installazione sloodle

L'installazione Sloodle contiene 3 cartelle separate. Queste devono essere installate in posti diversi della struttura di cartelle di Moodle.

  1. sloodle è un "modulo di attività" di Moodle ed è obbligatorio
  2. sloodle_menu è un "blocco" di Moodle
  3. sloodleobject è un "estensione di compiti" di Moodle

Installa il Modulo Sloodle

Moodle file directory as seen in Dreamweaver

Sloodle Versions: all (?)

Questa è la parte più importante di Sloodle -- è un modulo di attività che si installa dentro Moodle, lo trovate dentro la cartella "sloodle" che avete scaricato prima.

  1. Accedi alla tua installazione Moodle con FTP
  2. Apri la cartella "mod" e vedrai cartelle come assignment, chat, choice...
  3. Se c'è già una cartella chiamata "sloodle", cancellala
  4. Copia la cartella "sloodle" che hai trovato nello zip che hai scaricato
  5. Vai alla pagina di amministrazione del tuo browser web (aggiungi /admin alla url del tuo sito: http://miosito.org/moodle/admin)
  6. Vedrai una serie di azioni automatizzate che terminano con "sloodle tables have been set up correctly" (le tabelle di sloodle sono state installate correttamente)

Installa il menu di Sloodle (opzionale) e gli strumenti per i compiti (opzionale)

Il blocco di menu di Sloodle (Opzionale)

Sloodle Versions: 0.2, 0.21, 0.3 (?)

Questo è un componente opzionale che puoi usare per mettere un "blocco" nelle tue pagine Moodle. Quando lo hai impostato il blocco mostrerà il dettaglio del tuo avatar (se è conosciuto), insieme a diversi link utili a parti importanti parti di Sloodle. E' raccomandata l'installazione per facilitare la navigazione, ma non è essenziale. Puoi trovare i file nella cartella "sloodle_menu" che avevi scaricato precedentemente.

  1. Accedi alla tua installazione Moodle
  2. Apri la cartella "blocks"
  3. Se c'è già una cartella chiamata "sloodle_menu", cancellala
  4. Copia la nuova cartella "sloodle_menu"
  5. Vai alla pagina di amministrazione del tuo browser web (aggiungi /admin)

Plug_in per i compiti Sloodle Object (Opzionale)

Sloodle Versions: 0.3 (?)

Questo pluin è disponibile solo a partire dalla versione di Sloodle 0.3, e funziona come qualunque altro tipo di compito Moodle, è richiesto se vuoi usare lo strumento Sloodle PrimDrop. Sfortunatamente, è anche più complicato da installare. Devi trovare la cartella che si chiama "sloodleobject". Dovrebbe essere nella cartella originale che hai scaricato oppure, in alternativa, la trovi nella cartella "sloodle" (vedi la sezione Modulo Sloodle qui sopra), cercando in "sloodle/mod/primdrop-1.0".

  1. Accedi alla tua installazione Moodle
  2. Apri la cartella "mod/assignment/type"
  3. Se c'è già una cartella chiamata "sloodleobject", cancellala
  4. Copia la nuova cartella "sloodleobject"

Internazionalizzazione del plug-in dei compiti (Opzionale)

Cioè, hai installato il plugin dei compiti, ma potresti anche aver bisogno di aggiungere una riga ai file di traduzione. Probabilmente lo devi fare per ogni lingua installata nel tuo sito:

  1. Accedi alla tua installazione Moodle
  2. Apri la cartella "lang/en_utf8"
  3. Scaricati sul tuo pc il file "assignment.php" file (se sei connesso remotamente)
  4. Apri il file "assignment.php"
  5. Vai fino alla fine del file
  6. Subito dopo l'ultima linea "$string...", aggiungi una nuova riga: $string['typesloodleobject'] = 'Sloodle Object';
  7. Salva il file
  8. Invia il tuo file di nuovo al tuo server Moodle (via ftp), sovrascrivendo l'originale (se sei connesso remotamente)

Puoi ripetere i passi precedenti per gli altri linguaggi installati. Per esempio, se il tuo sito usa il francese, devi aprire nello step 2 la cartella "lang/fr_utf8".

Attiva gli strumenti

Vai alla pagina di amministrazione di Moodle (/admin) cliccando su "Notifiche" o "Amministrazione...". Puoi anche semplicemente aggiungere "/admin" all'indirizzo del tuo sito per attivare questa funzionalità. Questo assicura che Moodle finisca l'installazione, ed esegue tutti i cambiamenti al database.

Passi successivi

Sei adesso pronto a configurare Sloodle per il tuo sito, e poi prendere gli oggetti Sloodle.
