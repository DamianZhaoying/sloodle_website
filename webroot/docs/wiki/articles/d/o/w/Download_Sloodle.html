Download Sloodle

From SLIS Second Life Wiki

The SLOODLE software comes in two parts: the 'server-side' modules which you download and install in Moodle, and the 'in-world' tools which you obtain and use in Second Life or OpenSim. All of the SLOODLE software is completely free. That means it costs you nothing get it, and it is open source (released under the GNU General Public License version 3).

Version 2 (for Moodle 1.9, 2.0, 2.1, 2.2, 2.3, 2.4, 2.5)

This is the latest version of SLOODLE, for Moodle versions from 1.9 to 2.5. It is designed for Second Life and OpenSim.

Note: From SLOODLE 2 above, your Moodle server needs to be able to send HTTP messages to SL/OpenSim. For some hosts this requires adjusting firewall settings. See the network requirements page for more details.

1. Download the server-side modules:

  • SLOODLE 2.1 for Moodle 2.5 (zip) | (.tar)
  • SLOODLE 2.1 for Moodle 2.4 (zip) | (.tar)
  • SLOODLE 2.1 for Moodle 2.3 (zip) | (.tar)
  • SLOODLE 2.1 for Moodle 2.2 (zip) | (.tar)
  • SLOODLE 2.1 for Moodle 2.1 (zip) | (.tar)
  • SLOODLE 2.1 for Moodle 2.0 (zip) | (.tar)
  • SLOODLE 2.1 for Moodle 1.9 (zip) | (.tar)

If you prefer you can check out the code from the Git repository.

2. Get the in-world tools:

Second Life

  • Teleport to the SLOODLE area in Second Life - use a free Vending Machine to get the tools directly in Second Life.

OpenSim

  • SLOODLE 2.1 rezzer IAR for OpenSim

See the OpenSim documentation for instructions on how to load this into your grid.

3. Install:

After you have downloaded SLOODLE, please follow the installation instructions.

Version 1.2 (for Moodle 1.9)

This is the current stable release of SLOODLE, designed for Moodle 1.9. It was designed for Second Life, but an OpenSim version is maintained by the community.

1. Download the server-side modules:

  • Download SLOODLE *.zip - recommended for Windows users, or
  • Download SLOODLE *.tar.gz - recommended Linux/Mac OSX users

Advanced users may also be interested in the Add-ons and third-party extensions

If you prefer you can check out the code from the Subversion repository.

2. Get the in-world tools:

Second Life

  • Teleport to the SLOODLE area in Second Life - use a free Vending Machine to get the tools directly in Second Life.

OpenSim

A SLOODLE IAR is provided by the community. Please visit our OpenSim forums at sloodle.org for more information.

3. Install:

After you have downloaded SLOODLE, please follow the installation instructions.
